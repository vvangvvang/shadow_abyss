# Shadow Abyss — 暗影深渊

> 一款暗黑类 ARPG 地牢爬塔游戏，使用 Python + Pygame 构建。

---

## 游戏截图

游戏采用像素风俯视角地牢探索，每层包含随机生成的房间走廊、敌人、技能系统与等级成长。

---

## 运行方式

```bash
cd shadow_abyss
pip install pygame
python game.py
```

- Python 3.12+
- Pygame 2.6+

---

## 操作说明

| 按键 | 功能 |
|------|------|
| `W / ↑` | 向上移动 |
| `S / ↓` | 向下移动 |
| `A / ←` | 向左移动 |
| `D / →` | 向右移动 |
| `鼠标左键` | 选择目标敌人 |
| `1` | 技能1 — 火球（投射物） |
| `2` | 技能2 — 冰霜（AOE 范围伤害） |
| `3` | 技能3 — 冲锋（瞬间移动） |
| `4` | 技能4 — 治疗（恢复生命） |
| `E` | 进入下一层（在传送门附近） |
| `ESC` | 暂停 / 继续 |
| `R` | 游戏结束后重新开始 |

## 核心系统

### 🏰 地牢生成
- 每次进入新层随机生成 **10-15 个房间**，通过 L 形走廊连接
- 第一个房间中心为出生点，最后一个房间中心为传送门
- 每 5 层在传送门旁生成一只 **Boss**

### 🧭 流场寻路系统
- 使用 **BFS 广度优先搜索** 构建全场流场
- 每个可通行格子存储一个 **360 度单位向量**，指向距离目标最近的格子
- 玩家移动时流场实时更新，敌人追踪零开销
- （调试时可在地板上看到绿色方向箭头）

### 👁️ 敌人方向感系统
- 敌人获取方向后最多移动 **1 格距离**就需要重新感知
- 不在格子中心时优先回到中心再追踪，模拟"看不清远处"的行为
- 追踪路径使用流场，支持**任意角度平滑移动**

### 💥 碰撞反弹
- 敌人撞墙时自动沿法线方向选择可行走的新方向
- 比简单反转更自然，敌人不会卡在墙角

### ⚔️ 战斗系统
- **自动攻击**：选中敌人后，靠近自动普攻（5% 暴击率，150% 伤害）
- **4 个主动技能**：火球、冰霜（范围）、冲锋、治疗
- **等级成长**：击杀敌人获得经验，升级提升 HP 上限并回满血
- **掉落系统**：敌人死亡掉落金币和随机物品

### 🎨 视觉效果
- 棋盘格地板 + 立体感墙壁
- 玩家 4 方向 3 帧精灵动画（支持加载自定义 `res/player.png`）
- 敌人 4 帧精灵动画，Boss 体型翻倍（支持加载 `res/enemy.png`）
- 传送门呼吸动画
- 冷却遮罩、技能图标、UI 面板

## 敌人类型

| 类型 | 特征 | HP 倍率 | 伤害倍率 | 速度倍率 |
|------|------|---------|---------|---------|
| 🦴 Skeleton | 骷髅，基础属性 | 1.0× | 1.0× | 1.0× |
| 👺 Goblin | 哥布林，高攻高速 | 0.8× | 1.2× | 1.3× |
| 👹 Orc | 兽人，高血量 | 1.5× | 1.3× | 0.8× |

## 资源文件（可选）

将图片放入 `res/` 目录即可自动加载，否则使用纯色方块代替：

```
shadow_abyss/
└── res/
    ├── player.png      # 玩家精灵图（4行3列，每格32×32）
    ├── enemy.png       # 敌人精灵图（4行4列，每格32×32）

    ├── skill2.png
    ├── skill3.png
    └── skill4.png
```

## 配置参数

游戏常量可在 `game.py` 的 `CFG` 字典中调整：

```python
CFG = {
    'dungeon_w': 50,       # 地牢宽度（格）
    'dungeon_h': 40,       # 地牢高度（格）
    'rooms_min': 10,       # 最少房间数
    'rooms_max': 15,       # 最多房间数
    'room_min': 6,         # 房间最小尺寸
    'room_max': 12,        # 房间最大尺寸
    'player_speed': 4.0,   # 玩家移动速度（格/秒）
    'enemy_speed': 3.0,    # 敌人移动速度（格/秒）
    'aggro_range': 10,     # 敌人感知玩家距离（格）
}
```

# Shadow Abyss - 暗黑类 ARPG 游戏

![Python](https://img.shields.io/badge/Python-3.12+-blue)
![Pygame](https://img.shields.io/badge/Pygame-2.6+-green)

一款使用 Python + Pygame 开发的地下城探索游戏，包含完整的游戏机制。

---

## 🎮 游戏特性

### 核心系统
- **随机地牢生成** - 房间+走廊系统，每次开局都是全新的地图
- **流场寻路** - 360度任意方向寻路，敌人智能追踪玩家
- **方向感系统** - 敌人每隔1格重新寻路，模拟"看不清远处"的特性
- **碰撞反弹** - 敌人遇到墙壁时沿法线方向自然转向
- **中心点碰撞** - 简化的碰撞检测，避免卡墙

### 战斗系统
- **4个主动技能** - 火球(投射物)、冰霜(AOE)、冲锋(位移)、治疗(恢复)
- **自动攻击** - 靠近敌人自动攻击
- **暴击系统** - 5%暴击几率，150%伤害
- **升级系统** - 击杀敌人获得经验，升级提升属性

### 敌人类型
| 类型 | 特点 |
|------|------|
| 骷髅 (skeleton) | 基础敌人 |
| 哥布林 (goblin) | 高攻高速 |
| 兽人 (orc) | 高血量 |
| 恶魔 (demon) | 全面增强 |
| Boss | 每5层出现，超高属性 |

## 🔧 技术实现

### 流场寻路系统 (Flow Field)

```
核心思想：
1. 使用BFS计算每个格子到目标的距离
2. 每个格子存储一个指向"更近格子"的方向向量
3. 方向可以是任意角度（360度）
4. 敌人直接查询当前格子的方向
```

**优势**：
- 比A*更适合动态目标
- 更新代价小（只更新目标点）
- 支持任意角度平滑移动

### 方向感系统

```
工作原理：
1. 敌人获取方向后，可朝该方向移动1格距离
2. 移动1格后（方向感耗尽），重新获取方向
3. 如果不在格子中心，先回到中心
4. 碰撞时沿法线方向反弹
```

**效果**：
- 敌人不会直线追到天涯海角
- 碰撞后自然转向，不会死循环
- 更像"生物"的行为模式
